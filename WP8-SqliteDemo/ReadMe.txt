本项目仅仅针对 Windows Phone Silverlight 8.0,其他平台为测试

已经集成Sqlite 3.8.8.1到项目中
包含三个文件
Sqlite.wind
Sqlite.dll
sqlite3.dll

现在使用Sqlite不需要再加入其他工程进来

Sqlite.wind 封装源代码地址：https://github.com/KennethLK/sqlite-net-wp8
SQLite C#库安装
1 通过 NuGet UI安装: Tools->NuGet Package Manager->sqlite-net
2 通过NuGet Console安装： install-package sqlite-net

相关连接：
Sqlite SQL
http://sqlite.org/lang.html

Sqlite下载地址： http://sqlite.org/download.html
